# php-foundations-datastructures-Christian-Gabuya

Project Purpose

A complete book management system that organizes your library in three ways: by category for easy browsing, by search for quick finding, and alphabetically for simple reference.

Category Browser

Automatically scans all book categories and subcategories

Displays books in an expandable and collapsible folder structure

Shows the complete organizational hierarchy at a glance

Instant Search

Accepts any book title as input

Instantly retrieves stored details from the database

Displays author, publication year, and genre information

Clearly indicates whether the book is found or not found

Alphabetical System

Keeps all books sorted automatically

Removes the need for manual sorting

Provides a clean A-to-Z listing for easy scanning
